# Create APIs using Golang

This is a demo service used as an example for the "Create APIs using Golang" article series in my [blog](https://ssentinull.github.io/).

## How To Run

1. Run migration.

   ```shell
   $ make migrate-up
   ```

2. Copy paste `config.yml.example` to `config.yml`

3. Run server

   ```shell
   $ make run-server
   ```
